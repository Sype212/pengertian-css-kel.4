Transformasi CSS

Transformasi pada CSS digunakan untuk memanipulasi tampilan elemen seperti memperbesar/memperkecil (scale), memutar (rotate), dan memindahkan posisi (translate) tanpa mengubah dokumen aslinya.

Fungsi Penjelasan
scale() Memperbesar atau memperkecil ukuran elemen.
rotate() Memutar elemen sesuai derajat yang diinginkan.
translate() Memindahkan posisi elemen secara horizontal dan/atau vertikal.

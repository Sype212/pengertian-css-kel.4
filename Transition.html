Transition

Transisi CSS adalah efek perubahan nilai properti CSS secara halus (smooth) dalam rentang waktu tertentu, bukan berubah secara langsung. Transisi sering digunakan untuk membuat animasi sederhana saat elemen berubah keadaan, misalnya saat di-hover atau di-klik.

Properti Transition Keterangan
transition-property Menentukan properti CSS mana yang akan dianimasikan.
transition-duration Lama waktu transisi (misal: 0.5s).
transition-timing-function Jenis kecepatan transisi (ease, linear, dll).
transition-delay Waktu tunda sebelum transisi dimulai.
transition Shorthand untuk semua properti di atas.
